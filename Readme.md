## Installation

Npm:
```sh
npm i @anishshobith/deeptype
```

Yarn:
```sh
yarn add @anishshobith/deeptype
```

## Usage

JavaScript:
```js
const { Type } = require('@anishshobith/deeptype');

const type = new Type(40);

console.log(type.toString()); // number
```

TypeScript:
```js
import { Type } = from '@anishshobith/deeptype';

const type = new Type(40);

console.log(type.toString()); // number
```
